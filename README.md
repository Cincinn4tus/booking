# Lancer le projet

## Installer mysql avec un utilisateur 'root'@'localhost' avec le mot de passe 'root'
## Dans MySQL, lancer la commande
>`source lien_vers_le_fichier_script.sql`

## Lancer le serveur 

Sur Windows : `go run .\cmd\web\`
Sur Linux / MacOS : `go run ./cmd/web/`
